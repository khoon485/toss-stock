## 추천 기준

| 점수 | 추천 |
|------|------|
| ≥ 5 | STRONG_BUY |
| ≥ 4 | BUY |
| -3 ~ +4 | HOLD |
| ≤ -3 | SELL |
| ≤ -5 | STRONG_SELL |

## 이론적 범위

- **최고점수**: 약 +15~20점 (RSI 극단적 과매도 + 골든크로스 + 빗각 최적 + 콤보 보너스 등 모두 발동 시)
- **최저점수**: 약 -10~15점 (데드크로스 + 과매수 + 급락 등 모두 발동 시)

실제로는 대부분 **-5 ~ +8점** 범위에서 나옵니다.
